COLA – LEITURA E GRAVAÇÃO DE ARQUIVOS NO PROJETO

================================================================

1.  IMPORTAR CAMINHOS DO PROJETO

    from ds_core.paths import ( RAW_DATA_DIR, INTERIM_DATA_DIR,
    PROCESSED_DATA_DIR, EXTERNAL_DATA_DIR, FIGURES_DIR, REPORTS_DIR )

    import pandas as pd

================================================================

2.  LER ARQUIVOS EM data/raw

    CSV

         df = pd.read_csv(RAW_DATA_DIR / "arquivo.csv")

    CSV dentro de pasta

         df = pd.read_csv(
             RAW_DATA_DIR / "estatistica_descritiva" / "ifood.csv"
         )

    Excel

         df = pd.read_excel(RAW_DATA_DIR / "arquivo.xlsx")

    Parquet

         df = pd.read_parquet(RAW_DATA_DIR / "arquivo.parquet")

================================================================

3.  SALVAR DATASET EM data/interim (dados parcialmente tratados)

    df.to_csv( INTERIM_DATA_DIR / “dados_tratamento_inicial.csv”,
    index=False )

    ou

    df.to_parquet( INTERIM_DATA_DIR / “dados_tratamento_inicial.parquet”
    )

================================================================

4.  SALVAR DATASET EM data/processed (dados prontos para análise ou
    modelagem)

    df.to_csv( PROCESSED_DATA_DIR / “dataset_final.csv”, index=False )

    ou

    df.to_parquet( PROCESSED_DATA_DIR / “dataset_final.parquet” )

================================================================

5.  SALVAR FIGURAS EM outputs/figures

    import matplotlib.pyplot as plt

    plt.savefig( FIGURES_DIR / “grafico.png”, dpi=300,
    bbox_inches=“tight” )

================================================================

6.  SALVAR RELATÓRIOS OU TABELAS EM outputs/reports

    df.describe().to_csv( REPORTS_DIR / “estatisticas_descritivas.csv” )

    ou

    df_resultado.to_excel( REPORTS_DIR / “analise_final.xlsx”,
    index=False )

================================================================

7.  CRIAR PASTAS AUTOMATICAMENTE (CASO NÃO EXISTAM)

    (PROCESSED_DATA_DIR).mkdir( parents=True, exist_ok=True )

    exemplo completo

         output_path = PROCESSED_DATA_DIR / "dataset_final.csv"

         output_path.parent.mkdir(
             parents=True,
             exist_ok=True
         )

         df.to_csv(output_path, index=False)

================================================================

8.  FLUXO MAIS COMUM NO DIA A DIA

    1)  Ler dataset

        df = pd.read_csv(RAW_DATA_DIR / “dataset.csv”)

    2)  Limpar / transformar

    3)  Salvar dataset intermediário

        df.to_parquet( INTERIM_DATA_DIR / “dataset_limpo.parquet” )

    4)  Salvar dataset final

        df.to_parquet( PROCESSED_DATA_DIR / “dataset_final.parquet” )

    5)  Salvar gráficos

        plt.savefig( FIGURES_DIR / “grafico.png” )

================================================================

REGRA RÁPIDA PARA LEMBRAR

    raw
        dados originais

    interim
        dados em transformação

    processed
        dados prontos

    outputs/figures
        gráficos

    outputs/reports
        tabelas e resultados finais
